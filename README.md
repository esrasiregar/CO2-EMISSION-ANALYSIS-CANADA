## Langkah Instalasi
### Clone Repository
```
git clone https://github.com/esrasiregar/CO2-EMISSION-ANALYSIS-CANADA.git
cd CO2-EMISSION-ANALYSIS-CANADA
```

### Buat Virtual Environment
```
python -m venv .env
.env\Scripts\activate
```

### Install Dependensi
```
pip install -r req.txt
```

### Jalankan API
```
uvicorn app:app --reload
```
